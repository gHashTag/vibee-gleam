# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2024-12-23

### Added
- MCP server with 150+ tools for AI assistants
- Telegram MTProto integration via Go bridge
- P2P trading system (TON, xRocket, CryptoBot)
- RAG search with pgvector (hybrid vector + keyword)
- AI integrations (FLUX, Kling, ElevenLabs, Hedra)
- Video rendering service (Remotion)
- Task management system
- Multi-session Telegram support

### Security
- gitleaks pre-commit hooks for secret detection
- GitHub Actions secret scanning
- API key authentication for all endpoints
